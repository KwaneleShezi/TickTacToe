# Tic-Tac-Toe

A classic two-player Tic-Tac-Toe game built in Java with a graphical user interface.

---

## About

Two players take turns clicking on a 3x3 grid to place their marks (X or O). The game automatically detects wins and draws, and allows players to restart without closing the application.

---

## Features

- GUI built with Java Swing
- Two-player mode (local)
- Automatic win and draw detection
- Restart functionality after each game
- X and O image icons displayed on the board

---

## Built With

- Language: Java
- UI Framework: Java Swing
- IDE: Eclipse
